Translations for Liferay .
--------------------------------

Easiest way to start translating is to download Popeye from
http://popeye.sourceforge.net and open the project.ppf file.

Popeye is a GUI tool that helps in creating localization bundles.
It loads the bundles and shows the values side by side. It also 
highlights the missing translations.

Notice! Language_en.properties is a reference file for the translations. 
It should not be modified. The translations follow Liferay SVN trunk. 
If you want to contribute send your diff against latest version of 
Language_fi.properties to mika@javaguru.fi be sure to include your name
and email address you want to be added as a contributor to the file head.


Käännöksen tekeminen Liferayhin
----------------------------------

Helpoin tapa alkaa tekemään käännöstä on ladata Popeye 
http://popeye.sourceforge.net ja avata sillä porjet.ppf tiedosto.

Popeye on graafinen käyttöliittymä javan resurssitiedostojen kääntämiseen. 
Se näyttää alkuperäisen tekstin ja käännöksen rinnakkain. Puuttuvat käännökset
on helppo huomata sillä ne on korostettuna.

Huom! Language_en.properties tiedosto on vain referenssinä käännökselle. Sitä ei
tule muokata sillä se on vain kopio Liferay:n versionhallinnasta ja se päivitetään 
aika ajoin. Jos haluat parantaa käännöksiä tai lisätä kääntämättömiä tekstejä lähetä 
diffi viimeisintä Language_fi.properties tietostoa vasten osoitteeseen kaannokset@liferay.fi
Kerro myös nimesi ja sähköpostiosoitteesi jos haluat nimesi kääntäjä listaan.
